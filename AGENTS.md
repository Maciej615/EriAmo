# EriAmo - AI_Union Context

## Project Scope & Location
**CRITICAL:** The core logic of this project resides EXCLUSIVELY in the `AI_Union/` directory.
- Do not modify files outside of `AI_Union/` unless explicitly instructed.
- All analysis and refactoring should focus on `AI_Union/`.

## Project Description
Ten projekt implementuje "System Świadomość".
**Definicja:** Zdolność systemu do sterowania samym sobą w oparciu o model otoczenia.

## User Values (Maciek)
1. **Dążenie do ładu:** Priorytetem jest porządek w strukturze plików wewnątrz `AI_Union`.
2. **Autentyczność:** Rozwiązania proste, zrozumiałe, bez "magii".
3. **Uczciwość:** Jasne komunikowanie ograniczeń w komentarzach.

## Coding Standards (Python/C++)
- **Working Directory:** All paths should be relative to `AI_Union/`.
- **ADHD Friendly:** Kod musi być modularny. Jedna funkcja = jedno zadanie. Jasne nazwy zmiennych (self-explanatory).
- **Documentation:** Każdy kluczowy moduł w `AI_Union` musi mieć docstring wyjaśniający jego rolę w "Systemie Świadomość".
